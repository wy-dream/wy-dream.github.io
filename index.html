VuePress 为每个页面预渲染生成静态的 HTML，同时在页面被加载的时候，将作为 SPA 运行。

# vuepress博客搭建步骤

1

mkdir 项目名字(wzyBlog)

2通过npm来初始化项目

注意：在初始化之前要先进入到刚刚创建得这个文件

npm init -y

3.在wzyBlog下面创建一个docs文件夹 这个文件夹用来存放博客

mkdir docs

4.在docs文件夹下面，创建一个README.md得文件 输入以下内容

---
home: true
heroImage: /hero.png
actionText: 快速上手 →
actionLink: /zh/guide/
features:
- title: 简洁至上
  details: 以 Markdown 为中心的项目结构，以最少的配置帮助你专注于写作。
- title: Vue驱动
  details: 享受 Vue + webpack 的开发体验，在 Markdown 中使用 Vue 组件，同时可以使用 Vue 来开发自定义主题。
- title: 高性能
  details: VuePress 为每个页面预渲染生成静态的 HTML，同时在页面被加载的时候，将作为 SPA 运行。
footer: MIT Licensed | Copyright © 2018-present Evan You
---

5.在package.json里面 添加两条配置

    "dev":"vuepress dev docs",
    "build":"vuepress build docs"

6.创建一个.vuepress得文件夹

mkdir .vuepress

7.进入到这个.vuepress文件夹 然后创建一个config.js配置文件

mkdir .vuepress
cd .vuepress

8.在config.js里面增加配置代码

module.exports={
    title:"前端学习总结博客"
}

9.运行 回到项目根目录(wzyBlog)执行下面命令

vuepress dev docs

npm run deploy(写完后更新github的命令)

10.总结

注意: / -->指向docs目录 http:localhost:xxx/ 默认去访问README.md文件
