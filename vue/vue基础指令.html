# vue基础指令

a . v-on → 事件绑定,可以绑定大部分得事件 简写 :@click=" ", 如下：

    <div id="app">
        <button v-on:click="fn">点击按钮</button>
         <p v-if="flag">{{message}}</p>
         <p v-show="flag">{{message}}</p>
    </div>

    <script>
        new Vue({
            el: "#app",
            data: {
                message: "<h1>hello world<h1>",
                flag: true
            },
            methods: {
                fn() {
                    this.flag = !this.flag//取反
                }
            }
        })
    </script>

b . v-if → 相当于cs5里面的if语句，如果什么的，如下：

    <div id="app">
        <button v-on:click="fn">点击按钮</button>
         <p v-if="flag">{{message}}</p>
    </div>

    <script>
        new Vue({
            el: "#app",
            data: {
                message: "<h1>hello world<h1>",
                flag: true
            },
            methods: {
                fn() {
                    this.flag = !this.flag//取反
                }
            }
        })
    </script>

c . v-show → 可以理解为css样式里面的dispaly隐藏显示属性的切换，v-show只是单纯的切换css的display属性，内容始终都会被创建出来的，和if条件渲染不同，if条件渲染是会销毁内容的，如下：

    <div id="app">
        <button v-on:click="fn">点击按钮</button>
        <p v-if="flag">{{message}}</p> <!--彻底删除-->
         <p v-show="flag">{{message}}</p> <!--设置display属性-->
    </div>

        <script>
        new Vue({
            el: "#app",
            data: {
                message: "<h1>hello world<h1>",
                flag: true
            },
            methods: {
                fn() {
                    this.flag = !this.flag//取反
                }
            }
        })
    </script>

d . v-html → 等同于es5里面的innerhtml()，但在这里会把html标签解析显示在页面上，如下：

   <div id="app">
        <button v-on:click="fn">点击按钮</button>
        <p v-html="message"></p>
    </div>

    <script>
        new Vue({
            el: "#app",
            data: {
                message: "<h1>hello world<h1>",
                flag: true
            },
            methods: {
                fn() {
                    this.flag = !this.flag//取反
                }
            }
        })
    </script>

e . v-text → 等同于es5里面的innerText()方法，但在这里是把 Vue里面data里面的数据现在在标签里面，并不会向v-html一样，会解析html标签，v-text == {{}}模板语法，如下：

   <div id="app">
        <button v-on:click="fn">点击按钮</button>
        <p v-text="message"></p>
    </div>

    <script>
        new Vue({
            el: "#app",
            data: {
                message: "<h1>hello world<h1>",
                flag: true
            },
            methods: {
                fn() {
                    this.flag = !this.flag//取反
                }
            }
        })
    </script>

f . v-for → 等同于es5里面的for循环语句,需要使用item in items这种特殊的语法，items表示一个数据源数组，item表示每一个被循环出来的项,如下：

    <div id="app">
        <button @click="fn">点击按钮</button>
        <ul v-if="flag">
            <li v-for="item in arrlist"><span>{{item.username}}</span><span>：</span><span v-text="item.age"></span></li>
        </ul>
    </div>

    <script>
        new Vue({
            el: "#app",
            data: {
                flag: false,
                arrlist: [
                    { username: "小强", age: 19 },
                    { username: "小东", age: 11 },
                    { username: "小黑", age: 16 },
                    { username: "小红", age: 14 },
                    { username: "小兰", age: 13 }
                ]
            },
            methods: {
                fn() {
                    this.flag = !this.flag
                }
            }
        })
    </script>

注意1：可以循环对象

<div v-for="(value, key) in object">
{{ key }}: {{ value }}
</div>
<!-- 支持第三个参数 索引 -->
<div v-for="(value, key, index) in object">
{{ index }}. {{ key }}: {{ value }}
</div>

注意2：v-for是可以打印出数组元素的下标的,需要写成下面这样，第二个参数index就表示下标

<ul v-for="(item, index) in arr">
     <li>{{item}}-{{index}}</li>
</ul>

注意3：可以控制循环次数

<ul v-for="n in 10">
  <li>{{n}}</li>
</ul>

g . v-else-if → 也是相当于es5里面的else-if

<div id="app">
        <button v-on:click="fn">点击按钮</button>
        <br>
        <p v-if="num>=90">优秀</p>
        <p v-else-if="num>=80">良好</p>
        <p v-else-if="num>=70">良</p>
        <p v-else-if="num>=60">及格</p>
        <p v-else>不及格</p>
    </div>

    <script>
        new Vue({
            el: "#app",
            data: {
                num: 0
            },
            methods: {
                fn() {
                    //生成一个随机分数   floor向下取整 [0,101)
                    this.num = Math.floor(Math.random() * 101)
                    console.log(this.num)
                }
            }
        })
    </script>

h . v-bind → 用来绑定样式属性的，可以简写成：:属性名称="变量名"，如下：

   <div id="app">
        <a v-bind:href="href">百度</a> <!--简写 ：href-->
        <!-- 绑定src属性 -->
        <img :src="imageSrc" alt="">
        <img :src="'images/banner1.jpg'">
    </div>

    <script>
        new Vue({
            el: "#app",
            data: {
                href:"http://www.baidu.com"
            }
        })
    </script>

i . v-model → 创建双向绑定的数据，就是在一端改数据另一端也随之改变，如下：

   <div id="app">
        <input type="text" v-model.number="num">
        <button @click="fn">按钮</button>
        <p>{{num}}</p>
        <input type="radio" name="nan" value="男" v-model="rad">男
        <input type="radio" name="nv" value="女" v-model="rad">女
        <textarea name="" id="" cols="1" rows="1">{{rad}}</textarea>

        <hr>
        <select v-model="slelet">
        <option>打篮球</option>
        <option>打排球</option>
        <option>兵乓球</option>
        <option>羽毛球</option>
        </select>
        <textarea name="" id="" cols="15" rows="3">{{slelet}}</textarea>
    </div>
    <script>
    new Vue({
        el:"#app",
        data:{
            num:10,
            rad:"11",
            slelet:"11"
        },
        methods:{
            fn(){
                this.num++
            }
        }
    })
    </script>

j . v-pre → 限制元素不会被编译

<span v-pre>{{ message }}</span>

k . v-cloak → 在有很多想要在一定的时间后才显示的html标签时，就使用cloak属性，使用cloak后就可以控制所有的标签在一定的时间过后才显示，如下：

<style>
    [v-cloak]{
        display: none;
    }
</style>

<div id="app">
    <div v-cloak>
       <p>{{message}}</p>
       <p>{{message}}</p>
       <p>{{message}}</p>
       <p>{{message}}</p>
   </div>
</div>

<script>
    setTimeout(() => {
        let vm = new Vue({
            el: '#app',
            data: {
                message: 'hello nodeing'
            }
        });
    }, 3000)

</script>

l . v-once → 只渲染一次

<div id="app">
    <button @click="fn">点击</button>
    <p v-once>{{message}}</p>
</div>
<script src="node_modules/vue/dist/vue.js"></script>
<script>

    let vm = new Vue({
        el: '#app',
        data: {
            message: 'hello nodeing'
        },
        methods:{
            fn(){
                this.message = 'hello world!!!!'
            }
        }
    });
</script>

# for循环数组里的变异方法

变异方法就是指执行后原来的数组被改变 非变异方法就是指执行后原来的数组没有被改变： 变异方法：

a .push() 往数组最后添加一个元素

let arr=[1,333,4,5,6]
this.arr.push(111)

b .pop() 删除数组最后的一个元素

let arr=[1,333,4,5,6]
this.arr.pop()

c .shift() 删除数组的第一个元素

let arr=[1,333,4,5,6]
this.arr.shift()

d .unshift() 在数组的第一个位置添加元素

let arr=[1,333,4,5,6]
this.arr.unshift(1111)

e .splice() 可以删除或者添加元素 第一个参数表示删除或添加元素的开始位置 第二个参数表示删除多少个 第三个参数开始表示添加的元素

let arr=[1,333,4,5,6]
this.arr.splice(1,3,123,6,43)

f .sort() 排序 里面需要传入一个排序规则（函数）默认的是按照字符编码进行排序

升序

let arr=[1,333,4,5,6]
this.arr.sort((a,b)=>{
    return a > b
})

降序

let arr=[1,333,4,5,6]
this.arr.sort((a,b)=>{
    return a < b
})

g .reverse() 反转顺序

let arr=[1,333,4,5,6]
this.arr.reverse()
